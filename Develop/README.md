<h3 id='description'>Description of Project</h3>
<p>This project prompts a user through a series of questions designed to discover details about their project, then compiles it all into a good README.md file suitable for GitHub. I am not very at writing READMEs. Having a templated format with pre-written questions that I can answers reduces the stress of writing one. This project is great for people who don't know where to start on their README or who are looking for a way to write one quicker. I learned a lot about promises and async functions working on this project. This projet stands out due to the clean code on the backend and the funtional styling on the front end. This project features a series for prompts for the questionsand ansychronous processing.</p><br>

<h3 id='use'>How to Use the Project</h3>
<p>To use the project, open the folder and then open the index.js file. In terminal type in node index.js to run the program. Users will type in their responses or choose from a list using the spacebar to make their choices. Once all of the questions are answered, the user will push the READMD.md file to GitHub using the typical git commands ( add ., commit, push).</p><br>
